# 🥐 Crust & Crumb – Bakery Website

A modern bakery website designed with a warm European aesthetic.  
This project showcases a responsive layout, elegant typography, and a clean UI for an online bakery experience.

---

## 🌟 Features

- Elegant bakery themed design
- Responsive layout
- Interactive order form
- Menu showcase
- Smooth scrolling navigation

---

## 🛠️ Built With
